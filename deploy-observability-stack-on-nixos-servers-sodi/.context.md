# Goal
Metrics and alerting for NixOS infrastructure.

# Components
- Prometheus: Metrics collection
- Grafana: Dashboards
- Alertmanager: Alert routing
- Node exporter: System metrics

# Acceptance Criteria
- [ ] Prometheus scraping all NixOS hosts
- [ ] Grafana with infrastructure dashboards
- [ ] Alertmanager routing to Matrix/Discord
- [ ] Basic alerts: disk, memory, service down

# Files to modify
- modules/nixos/prometheus.nix (may exist, check)
- modules/nixos/grafana.nix (new or extend)
- modules/nixos/alertmanager.nix (new)

# Notes

Consider federation if scaling to multiple Prometheus instances.
