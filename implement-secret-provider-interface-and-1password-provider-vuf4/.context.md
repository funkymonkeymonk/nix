# Goal
Create the pluggable provider system and implement 1Password Connect provider.

# Acceptance Criteria
- [ ] Provider interface defined in traverse/pkg/provider/
- [ ] 1Password Connect provider implemented
- [ ] Provider registry for dynamic loading
- [ ] Health check for provider connectivity
- [ ] Support for listing secrets (optional)
- [ ] Proper error handling and retry logic

# Files to Create
- traverse/pkg/provider/provider.go (interface definition)
- traverse/pkg/provider/registry.go (provider registry)
- traverse/pkg/providers/1password/1password.go (1Password implementation)
- traverse/pkg/providers/1password/config.go (1Password config)
- traverse/pkg/providers/local/local.go (local file provider - future)

# Key Implementation Details
- Use 1Password Connect SDK if available, else REST API
- Support for service account tokens
- Caching layer to avoid repeated Connect API calls
- Timeout and retry configuration
