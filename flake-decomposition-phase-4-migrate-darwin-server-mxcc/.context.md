# Problem
darwin-server is a headless Darwin machine (targets/darwin-server/default.nix, 454 lines). It's complex — Lume VM management, Ollama with Metal, OpenClaw/Wadsworth, cloud-init support, plist workarounds. Needs a parallel v2 output using library.lib.mkDarwinSystem + library/archetypes/headless-server-darwin.nix.

This is the first Darwin migration and the most complex single-target file in the repo.
