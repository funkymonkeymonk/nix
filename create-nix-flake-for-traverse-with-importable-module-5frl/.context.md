# Goal
Create a Nix flake at the root of ~/src/funkymonkeymonk/traverse/ that:
1. Defines the Traverse package (Go binary)
2. Provides importable NixOS and Darwin modules
3. Supports both x86_64-linux and aarch64-darwin
4. Can be imported by your main nix repo's flake.nix

# Acceptance Criteria
- [ ] traverse/flake.nix exists with proper inputs/outputs
- [ ] traverse/packages/traverse.nix builds the Go binary
- [ ] traverse/nixos-modules/traverse.nix provides NixOS service module
- [ ] traverse/darwin-modules/traverse.nix provides nix-darwin service module
- [ ] traverse/lib/traverse-options.nix exports options schema for type checking
- [ ] Can be imported in main flake: `inputs.traverse.url = "path:./traverse";`
- [ ] Supports both NixOS and Darwin deployment targets

# Files to Create
- traverse/flake.nix (main flake definition)
- traverse/packages/traverse.nix (Go build)
- traverse/nixos-modules/default.nix (NixOS module entry)
- traverse/nixos-modules/traverse.nix (NixOS service config)
- traverse/darwin-modules/default.nix (Darwin module entry)
- traverse/darwin-modules/traverse.nix (Darwin service config)
- traverse/lib/traverse-options.nix (shared options schema)

# Key Implementation Details
- Service should run as systemd service (NixOS) or launchd service (Darwin)
- Configuration via environment variables and config file
- Support for loading credentials from agenix/ragenix or sops-nix
- Health check endpoint for service monitoring
- Log rotation configuration
