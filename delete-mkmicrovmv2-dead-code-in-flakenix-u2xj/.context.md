# Notes
The actual v2 MicroVMs (`dev-vm-v2`, `openclaw-v2`, etc.) call `libraryLib.mkNixosSystem` directly. `_mkMicrovmV2` was an intermediate helper that was superseded before it was ever wired up. The underscore prefix indicates it was intended as private, but it should be deleted rather than left as dead code.
