# Goal
Build the notification system that sends approval requests via Duo Push and Slack.

# Acceptance Criteria
- [ ] Notification provider interface
- [ ] Duo Push provider with interactive approve/deny
- [ ] Slack provider with block kit buttons
- [ ] Notification manager for multi-channel delivery
- [ ] Callback handlers for interactive responses
- [ ] Notification templates with context

# Files to Create
- traverse/pkg/notification/notification.go (interface)
- traverse/pkg/notification/manager.go (multi-provider logic)
- traverse/pkg/notifications/duo/duo.go (Duo implementation)
- traverse/pkg/notifications/slack/slack.go (Slack implementation)
- traverse/pkg/notifications/webhook/webhook.go (generic webhook)
- traverse/internal/callback/server.go (callback HTTP handlers)

# Key Implementation Details
- Duo: Use duo_api_golang library
- Slack: Use slack-go/slack library
- Support notification priority levels
- Rate limiting per provider
- Failed notification retry with backoff
