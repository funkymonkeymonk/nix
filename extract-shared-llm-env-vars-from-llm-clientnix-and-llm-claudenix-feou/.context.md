# Problem
NOTE: The yak name has wrong file names. The actual files are claude.nix, opencode.nix, pi.nix.

`LLM_SERVER_HOST` and `LLM_SERVER_PORT` are set identically in three role files:
- modules/roles/claude.nix lines 28-29
- modules/roles/opencode.nix lines 27-28
- modules/roles/pi.nix lines 31-32

Additionally, claude.nix and opencode.nix both set:
- `OPENCODE_ENDPOINT = "http://${host}:${port}";`
- `CLAUDE_API_BASE = "http://${host}:${port}";`

# Solution
Create `modules/common/llm-client.nix` that:
1. Reads `myConfig.llmClient.serverHost` and `myConfig.llmClient.serverPort`
2. Sets `LLM_SERVER_HOST`, `LLM_SERVER_PORT`, `OPENCODE_ENDPOINT`, `CLAUDE_API_BASE`
3. Is enabled when any of the AI agent roles are active (claude, opencode, pi)
4. Import it in modules/common/default.nix (or wherever common modules are imported)
5. Remove the duplicate env var blocks from claude.nix, opencode.nix, pi.nix
