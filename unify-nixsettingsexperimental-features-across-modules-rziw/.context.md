# Goal
Consolidate duplicate nix.settings.experimental-features = ["nix-command" "flakes"] definitions.

# Background
os/nixos.nix is only imported by the main flake for regular nixosConfigurations.
Standalone targets (bootstrap, installer-iso, machine-types) don't import os/nixos.nix —
they need their own copy for when deployed standalone.

# What can be cleaned up
- os/microvm.nix imports os/nixos.nix indirectly — if nixos.nix already sets it, os/microvm.nix duplication can be removed
- machine-types/server.nix and machine-types/desktop.nix — check if these are used in configs that also include os/nixos.nix

# Acceptance Criteria
- [ ] Identify which duplicates are genuinely redundant (same config path includes os/nixos.nix)
- [ ] Remove redundant copies (at minimum os/microvm.nix if nixos.nix sets it)
- [ ] Standalone configs that don't inherit from os/nixos.nix keep their own copy
- [ ] check:lint and test:nixos-eval pass

# Files
- os/nixos.nix (canonical, already has it)
- os/microvm.nix (likely redundant)
- machine-types/server.nix (check if redundant)
- machine-types/desktop.nix (check if redundant)
- targets/bootstrap/default.nix (standalone, keep)
- targets/installer-iso/default.nix (standalone, keep)
