# Goal
Move 1Password Connect from protoman (Darwin) to a NixOS server for better reliability.

# Rationale
Running Connect on Darwin headless server has issues:
- Docker Desktop requires GUI session
- Podman needs VM setup (lima) which has compatibility issues
- Colima cannot run as root
- Manual start required after reboot

NixOS provides native systemd service support for containers.

# Architecture After Move
```
protoman (Darwin)          NixOS Server
├─ MicroVMs ─────────────► ├─ 1Password Connect
│  (fetch secrets)         │  (systemd + podman/docker)
└─ opnix (client) ───────► └─ Serves secrets to all MicroVMs
```
