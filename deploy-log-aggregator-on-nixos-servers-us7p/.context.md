# Goal
Centralized logging for all NixOS infrastructure.

# Components
- Vector: Log shipper (systemd service)
- Loki: Log storage (NixOS service or container)

# Acceptance Criteria
- [ ] Vector service on all NixOS hosts shipping logs
- [ ] Loki instance receiving logs from all hosts
- [ ] Log labels: host, service, severity
- [ ] Query interface available

# Files to modify
- modules/nixos/vector.nix (new)
- modules/nixos/loki.nix (new)
- Add to relevant target configurations

# Notes
Consider using existing Loki on darwin-server as central aggregator,
or deploy separate Loki per environment (homelab/production).
