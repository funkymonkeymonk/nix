# Goal
The myConfig.vane.autoStart option (default: false) is defined in options.nix but never
consulted by modules/services/vane/darwin.nix, which always sets RunAtLoad = true.

# Acceptance Criteria
- [ ] launchd.user.agents.vane.serviceConfig.RunAtLoad uses cfg.autoStart value
- [ ] When autoStart = false (default), RunAtLoad = false
- [ ] When autoStart = true, RunAtLoad = true
- [ ] check:lint and test:darwin-eval pass

# Files
- modules/services/vane/darwin.nix (RunAtLoad = true → RunAtLoad = cfg.autoStart)
- modules/common/options.nix (reference for autoStart option location)
