# Problem
core (Darwin) and bootstrap (NixOS) are minimal configs used for initial system setup. They use only modules/common/core.nix and modules/common/options.nix. These are the simplest migrations — minimal modules, no user config, no services.

# Acceptance Criteria
- [ ] core-v2 added as parallel darwinConfiguration using library.lib.mkDarwinSystem
  - hostname = "core", system = "aarch64-darwin"
  - modules: [ ./modules/common/core.nix ]
  - nixpkgs.config.allowInsecurePredicate for openclaw/olm NOT needed (no openclaw)
  - system.stateVersion = 4
- [ ] bootstrap-v2 added as parallel nixosConfiguration using library.lib.mkNixosSystem
  - hostname = "nixos-bootstrap", system = "x86_64-linux"
  - modules: [ ./modules/common/core.nix, ./modules/common/options.nix, { nixpkgs.hostPlatform = "x86_64-linux"; system.stateVersion = "25.05"; } ]
  - Note: the old bootstrap config has hardware stub, filesystems, users, auto-apply — none of these use modules/common/core.nix exclusively; the existing flake.nix bootstrap definition at lines 308-319 only references core.nix, options.nix, targets/bootstrap
- [ ] Old darwinConfigurations.core and nixosConfigurations.bootstrap unchanged
- [ ] `nix store diff-closures .#core .#core-v2` is empty
- [ ] `nix store diff-closures .#bootstrap .#bootstrap-v2` is empty
- [ ] `devenv tasks run test:all` passes

# Notes
- mkNixosSystem automatically includes home-manager + opnix — for bootstrap, this may add unwanted complexity. Consider whether bootstrap-v2 should use mkNixosSystem or a simpler nixpkgs.lib.nixosSystem (since bootstrap is intentionally minimal)
- Similarly, mkDarwinSystem includes the full modules/ import. core should have minimal modules
- This tension between "use the library helper" and "these are intentionally minimal" needs resolution: either expand mk*System to accept a `minimal` flag, or use raw nixpkgs.lib.nixosSystem/nix-darwin.lib.darwinSystem with archetype flag
- The design doc says these are "minimal configs — low risk" — the v2 output should be as close to identical as possible
