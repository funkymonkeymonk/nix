# Problem
`modules/roles/entertainment.nix` only contains homebrew casks (steam, obs, discord) guarded by `lib.mkIf isDarwin`. On NixOS the role enables but installs nothing — it's a no-op.

The role should provide equivalent Linux packages when enabled on NixOS.

# Acceptance Criteria
- [ ] When `myConfig.roles.entertainment.enable = true` on NixOS, at least the core apps are available as packages
- [ ] Steam: use `programs.steam.enable = true` (NixOS has a dedicated module)
- [ ] OBS: add `pkgs.obs-studio` to `environment.systemPackages`
- [ ] Discord: add `pkgs.discord` to `environment.systemPackages`
- [ ] Darwin behavior unchanged (still uses homebrew casks)
- [ ] `devenv tasks run test:nixos-eval` passes
