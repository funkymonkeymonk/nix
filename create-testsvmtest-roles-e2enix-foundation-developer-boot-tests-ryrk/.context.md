# Goal
Create tests/vm/test-roles-e2e.nix with VM tests that boot a full-stack NixOS config
(using mkVmTest helper) and verify foundation + developer role behavior at runtime.

# Implementation Notes

## Use mkVmTest, NOT mkTest
mkTest (existing) is stripped-down — only options.nix + roles/default.nix + nixos/base.nix.
mkVmTest (new) imports full ./modules + home-manager + disko + inline facter stub.
See parent yak for full mkVmTest design and the inline facter stub pattern.

## Test cases to implement

### foundation-e2e
- Enable myConfig.roles.foundation.enable = true
- Boot and assert: hx --version, jj --version, rg --help, delta --version
- Assert: zsh is default shell for testuser
- Assert: 1password disabled (onepassword.enable = false) — no external deps

### developer-e2e
- Enable foundation + developer roles
- Boot and assert: node --version, python3 --version, gh --version, clang --version
- Assert: jq --version (dev tool)
