# Goal
Eliminate the duplicated nixpkgs configuration (overlays, allowUnfree, permittedInsecurePackages, allowInsecurePredicate) between `flake.nix` and `library/lib/mk-system.nix`.

## Prerequisites
- "Consolidate v1/v2 flake configurations" must be done

# Acceptance Criteria
- [ ] Extract the nixpkgs config block from `flake.nix` into a shared module, e.g. `modules/common/nixpkgs-config.nix`
- [ ] Refactor `library/lib/mk-system.nix` to import `modules/common/nixpkgs-config.nix` instead of inlining the config
- [ ] Refactor `flake.nix` `configuration` helper to also import the shared module
- [ ] Ensure the overlay list is defined in exactly one place
- [ ] Verify all flake outputs still evaluate correctly
- [ ] Run `devenv tasks run check:lint` and `devenv tasks run test`

# Files
- `flake.nix`
- `library/lib/mk-system.nix`
- `modules/common/nixpkgs-config.nix` (new)

# Notes
Both files currently contain identical overlays and config:
```nix
overlays = [
  (final: _prev: { stable = import inputs.nixpkgs-stable { inherit (final) system config; }; })
  (final: _prev: { inherit (inputs.devenv.packages.${final.stdenv.hostPlatform.system}) devenv; })
  (final: _prev: { zellij-pane-tracker = inputs.zellij-pane-tracker.packages.${final.stdenv.hostPlatform.system}.default; })
  (import ../../overlays { inherit inputs; })
];
```
This duplication means any overlay change must be made in two places, which is error-prone.
