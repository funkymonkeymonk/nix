# Writing Skills

## Overview

Writing skills for documentation, skill files, and technical content.

## When to Use

- Creating documentation
- Writing skill files
- Technical writing tasks

## Process

1. **Know your audience** - who will read this?
2. **Outline structure** - what sections are needed?
3. **Write clearly** - simple language, short sentences
4. **Use examples** - show, don't just tell
5. **Review and revise** - clarity over cleverness

## Key Principles

- Clear beats clever
- Structure aids comprehension
- Examples illustrate concepts
- Revise for clarity
