# Using Superpowers

## Overview

Access and use available skills for the current task.

## When to Use

- Starting any non-trivial task
- Unsure which skill applies
- Want to ensure best practices

## Process

1. **Identify the task type** - development, debugging, writing, etc.
2. **Check available skills** - what skills are installed?
3. **Select appropriate skill** - which matches your task?
4. **Apply the skill** - follow its process
5. **Iterate** - use multiple skills as needed

## Key Principles

- Always check for relevant skills first
- Skills can be combined
- If unsure, ask which skill to use
- Skills improve consistency and quality
