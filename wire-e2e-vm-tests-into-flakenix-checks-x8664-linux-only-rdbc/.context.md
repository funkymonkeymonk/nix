# Goal
Add all Tier 4 e2e VM tests to flake.nix checks.x86_64-linux so they are
discoverable by `nix flake check` and cacheable by Cachix.

# Implementation Notes

## Current pattern (from tests/default.nix)
```nix
vmTests =
  if isLinux && self != null
  then import ./vm {inherit pkgs self;}
  else {};
```
The existing vm/ tests are already imported this way. The new e2e tests follow the same pattern.

## What to add to tests/vm/default.nix

```nix
testRolesE2e    = import ./test-roles-e2e.nix    {inherit pkgs self;};
testServicesE2e = import ./test-services-e2e.nix {inherit pkgs self;};
```
