# Goal
Migrate all four microvms to the new library-based structure.
MicroVMs are dispensable (replaced not upgraded) making them the lowest-risk first target.

# Reference
Full design: docs/plans/2026-05-11-flake-decomposition-design.md (Section 11, Phase 1)

# Per-VM Pattern
For each of dev-vm, openclaw, matrix, media-center:
1. Add <vm>-v2 output to flake.nix using mkNixosSystem with archetype=microvm-guest
2. Build: nix build .#nixosConfigurations.<vm>-v2.config.system.build.toplevel
3. Diff closures: nix store diff-closures .#microvm.nixosConfigurations.<vm> .#nixosConfigurations.<vm>-v2
4. If diff is clean: switch runner to use v2
5. Delete old output, rename v2 → final name
